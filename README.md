# Expense Tracker
Python 3.12.3, Linux, CLI, JSON

## About This Project

A small expense tracker that can stores expense data in JSON and allows users to complete basic commands such as adding, deleting, and viewing their expenses. 

This project was developed in Python 3.12.3 and stores the user's expenses in JSON files.
